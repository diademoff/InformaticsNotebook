Общая информация

Зачем нужен этот проект?

Проект направлен на решение проблем качественной подготовки заданий для проверки знаний учащихся школьников средней школы за счёт генерации уникальных заданий, а так же возможности анализа статистики пройденных заданий для выявления сложностей в пройденном материале.

Кому полезен этот проект?

  • Учителям - быстрая и качественная проверка знаний ученика
  • Ученикам - выявление пробелов в знаниях
  • Учебному учреждению - возможность собирать и анализировать статистику учеников

Почему наш проект?

  • Обширный каталог заданий
  • Оффлайн программа
  • Индивидуальная генерация заданий
  • Бесплатная программа с открытым кодом

Быстрый старт

  • Создайте тест
  • Решите тест
  • Собирайте статистику о выполненных заданиях

Создание теста

Что такое тест?

Тест это файл в котором храниться информация о заданиях. Расширение имени файла .test

Как создать тест?

Тест может создать только учитель.

  • Поэтому заходим в программу как учитель.
  • Все задания разбиты на категории, выбирайте нужные вам задания.
  • Также вы можете указать количество одно и того же задания.
  • Настраиваем тест.
  • Сохраняем тест в удобный для вас каталог с любым именем.
  • Теперь можно решить его.

Выполнение заданий теста

  • Зайдите в программу как ученик.
  • Введите ФИО и класс или выберете из списка уже созданных учеников.
  • Нажмите на кнопку запуска теста.
  • Выберете файл теста.
  • После решения заданий или завершения теста вы увидите результат по этому тесту.

Просмотр статистики

Где хранится статистика?

Статистика хранится в файле по пути %userprofile%\documents\Notebook\ Поэтому вы можете посмотреть статистику выполнения тестов, если они выполнялись на этом компьютере.

Как смотреть результаты?

  • Зайдите в программу как учитель.
  • Выберете ученика, по которому хотите посмотреть статистику.
  • Смотрите информацию по каждому тесту отдельно

Сбор статистики с нескольких компьютеров

  • Скачайте программу Statistics.exe (включая dll и другие файлы)
  • Запустите Statistics.exe
  • Выберите папку с .bin файлами
  • Выберите нужных учеников и их тесты
  • Статистика сохранится в файл
